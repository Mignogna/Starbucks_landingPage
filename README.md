# **☕ Starbucks Coffee - Landing Page**

Este é um projeto de desenvolvimento de uma Landing Page responsiva para o Starbucks, desenvolvida com o objetivo de praticar conceitos fundamentais de Frontend.

**Nota:** Este projeto foi desenvolvido acompanhando o tutorial da [Larissa Kich](https://www.youtube.com/@LarissaKich) no YouTube. É um projeto para fins exclusivamente educacionais e de estudo de interface.


## **🚀 Tecnologias Utilizadas**

O projeto foi construído utilizando as seguintes ferramentas:

* **HTML5:** Estruturação semântica de seções.
* **CSS3:** Estilização com o uso de Variáveis CSS, Flexbox e CSS Nesting.
* **JavaScript:** Lógica para o menu hamburguer (mobile) e efeitos de rolagem.
* **Font Awesome:** Biblioteca de ícones para redes sociais e interface.
* **Google Fonts:** Tipografia personalizada (Poppins).

## **🧠 Aprendizados**

Durante o desenvolvimento deste projeto, pude reforçar e aprender novos conceitos:

* **Responsividade (Mobile First):** Ajuste de layout para diferentes tamanhos de tela usando `@media queries`.
* **Organização de Variáveis:** Uso do `:root` para manter uma paleta de cores consistente e fácil de dar manutenção.
* **Manipulação de DOM:** Criação de um menu interativo que abre e fecha via JavaScript.
* **UX/UI:** Implementação de `scroll-margin-top` para evitar que o header fixo cubra os títulos das seções ao navegar pelo menu.
* **Clean Code:** Estruturação de pastas (`assets`, `css`, `js`) para um projeto organizado.

# **☕ Starbucks Coffee - Landing Page**

This is a responsive Landing Page project for Starbucks, developed with the goal of practicing fundamental Frontend concepts.

**Note:** This project was developed following the tutorial by [Larissa Kich](https://www.youtube.com/@LarissaKich) on YouTube. It is intended exclusively for educational purposes and interface study.


## **🚀 Technologies Used**

The project was built using the following tools:

* **HTML5:** Semantic structuring of sections.
* **CSS3:** Styling using CSS Variables, Flexbox, and CSS Nesting.
* **JavaScript:** Logic for the hamburger menu (mobile) and scroll effects.
* **Font Awesome:** Icon library for social media and interface elements.
* **Google Fonts:** Personalized typography (Poppins).


## **🧠 Key Learnings**

During the development of this project, I was able to reinforce and learn new concepts:

* **Responsiveness (Mobile First):** Layout adjustment for different screen sizes using `@media queries`.
* **Variable Organization:** Using `:root` to maintain a consistent and easy-to-maintain color palette.
* **DOM Manipulation:** Creating an interactive menu that opens and closes via JavaScript.
* **UX/UI:** Implementation of `scroll-margin-top` to prevent the fixed header from covering section titles during navigation.
* **Clean Code:** Folder structuring (`assets`, `css`, `js`) for an organized project.
